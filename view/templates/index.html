Linked Data API

This is the home page of GA's Linked Data API of Samples, Sites and Surveys. View the registers for the three categories in the navigation bar at the top of the page.

Sample

This system provides landing pages for GA's samples, resolving with the use of International GeoSample Number identifiers. Multiple Linked Data views of each sample's metadata according to several relevant models such as the IGSN model, PROV-O and Dublin Core are available in this system.

There are two methods of access to GA's samples' metadata that this API delivers:

  1. Linked Data - human and machine readable resources (web pages & RDF data) for the Register of samples and each Sample
  2. OAI-PMH - Open Archives Initiative - Protocol for Metadata Harvesting: an HTTP API for harvesting metadata for objects in XML. This form of access is a requirement of the IGSN community

The endpoints for these two forms of access are given in the links at the top of the page in the navigation bar.

Site

This system provides landing pages for digital catalogue record sites utilised by GA. It contains views according to models such as the National Environmental Monitoring Sites Register and the Geoscience Australia's Public Data Model ontology.

Surveys

This system provides the landing pages for metadata about Surveys stored in GA's corporate database known as ARGUS. It contains views according to models such as GA'S ARGUS, GA'S Public Data Model, PROV-O and the Semantic Sensor Network Ontology.
